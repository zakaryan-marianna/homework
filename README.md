# Homework

## Requirements
1. PHP 8.1

## How to install
1. Clone the repository
2. Copy .env.example to .env
3. Run `composer install`

## How to use
1. Copy the input CSV file to `storage/app` folder.
2. Run `php artisan commissions:calculate $FILENAME.csv`

## How to run tests
Just run `php artisan test`.
